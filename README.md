# Adisty-uiux-Projects
ADISTY is a mobile application designed specifically for Ahmad Dahlan University students to simplify academic and financial management. The goal of this updated design is to facilitate tuition payments using a student e-card that can be integrated with UAD's partner banks.
Note: These screens are part of a larger project. For a full view of the interactive prototype, feel free to check the Figma link below.
